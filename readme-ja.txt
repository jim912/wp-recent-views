== Description ==
最近見たページのリストを表示します。 


== Installation ==
1. プラグインの新規追加メニューをクリックし、プラグインのインストール画面を表示します。
2. 検索欄に"WP Recent Views"と入力して検索します。
3. WP Recent Views の"いますぐインストール"リンクをクリックして、インストールを行います。
